Proyecto con Laravel

En este proyecto, lo que hemos utilizado es el framework que se llama el Laravel. Lo que hemos hecho es listar unos proyectos y estos también se pueden editar, eliminar y crear. En conclusión, hemos creado el crud completo de la aplicación.

Proyecto con una Api

En este proyecto, lo que hemos hecho ha sido atacar a una API que nos ha proporcionado el profesor, hemos echo diferentes peticiones al servidor donde se ubica la API y mediante esas peticiones lo que van a hacer es devolvernos un estado 200 si está todo perfecto o un estado 404 si la petición ha sido fallida.

Proyecto galería de fotos

En este proyecto, lo que hemos hecho es simular que teníamos una base de datos, que en nuestro caso van a hacer las fotos y lo que hemos hecho como resultado final es realizar una galería de fotos para los que son amantes de las fotografías y tengan un sitio donde poder ver de una forma más dinámica sus fotos.

PRW

Este es uno de mis proyectos más recientes, realizado como trabajo de fin de ciclo. En este proyecto, he tenido la oportunidad de desarrollar una API REST sencilla utilizando PHP, complementada con JavaScript, Bootstrap, CSS, HTML y SQL. Esta es la versión inicial de mi proyecto final, y en el futuro planeo mejorar varios aspectos, como el diseño, y añadir, o al menos intentar añadir, un apartado de autenticación con tokens.

Framework laravel

En este proyecto, hemos utilizado el framework Laravel. Hemos implementado la funcionalidad para listar, editar, eliminar y crear proyectos. En resumen, hemos desarrollado el CRUD completo de la aplicación.

Proyecto con una API

En este proyecto, hemos interactuado con una API proporcionada por el profesor. Realizamos diferentes peticiones al servidor donde está alojada la API. Estas peticiones devuelven un estado 200 si la operación fue exitosa o un estado 404 si la petición falló.

Galería de imágenes

En este proyecto, hemos simulado una base de datos utilizando fotos. El resultado final es una galería de fotos diseñada para los amantes de la fotografía, proporcionando un lugar donde puedan ver sus imágenes de una forma más dinámica y atractiva.
